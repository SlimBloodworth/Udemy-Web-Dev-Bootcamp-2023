Snowman Logo Exercise

Challenge: Find and use the snowman entity code and the registered trademark entity code to create a logo for "Snowman Productions". The snowman should be in front of the words and the registered trademark should be at the end of the words and should be in superscript. This exercise is to practice using and understanding entity codes, as well as superscript and subscript tags.
