Bootstrap Exercises

Bootstrap is a CSS framework which consists of pre-built components which are mainly accessed via classes, and a responsive grid system. Bootstrap is used to help more quickly build responsive web applications. To include Bootstrap in a project you can either download everything on to your machine, or link to a CDN. For more information visit getbootstrap.com.
